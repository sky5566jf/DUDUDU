📖 API 参考

GET/api/device - 设备信息 GET/api/status - 服务器状态 GET/api/clients - VNC 客户端列表 GET/api/screenshot - 屏幕截图 POST/api/input - 输入文本 POST/api/key?code=13 - 发送按键 POST/api/clipboard_text - 设置剪贴板 GET/api/clipboard_text - 获取剪贴板 POST/api/writefile_text?path=xxx - 写入文件 POST/api/upload?path=xxx - 上传文件 POST/api/reboot - 重启 POST/api/respring - 注销 POST/api/screen/lock - 锁屏 POST/api/screen/unlock - 解锁 POST/api/home - Home 键 POST/api/taskmanager - 任务管理器 GET/api/volume - 获取音量 POST/api/volume?value=0.5 - 设置音量 GET/api/brightness - 获取亮度 POST/api/brightness?value=0.5 - 设置亮度 POST/api/install?path=xxx.ipa - 安装应用 POST/api/uninstall?bundleId=xxx - 卸载应用 GET/api/trollstore/diagnostics - TrollStore 诊断 POST/api/clearapps/smart - 智能清理后台 GET/api/assistivetouch - 小白点状态 POST/api/assistivetouch?action=enable - 启用小白点 POST/api/assistivetouch?action=disable - 禁用小白点 GET/api/trigger?port=3333&delay=5 - 触发懒人精灵
