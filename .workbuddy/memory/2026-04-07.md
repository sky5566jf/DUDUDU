## TrollVNC 新增设备保存 API

### 功能说明
- `GET /api/device?ip=xxx&save=true` - 获取设备信息并保存到手机
- 保存路径：`/var/mobile/Media/fuwuduan.txt`
- 保存内容包括：deviceId、deviceModelName、systemName、deviceName、serverIP、deviceModel、recordTime、systemVersion、batteryLevel、batteryState
