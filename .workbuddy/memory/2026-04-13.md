## API 修改

### /api/device 新增存储信息
- 在 `TVNCApiManager.mm` 的 `-getDeviceInfo` 方法中添加了 storage 字段
- 返回 totalBytes/freeBytes/usedBytes 及换算后的 GB 值

### /api/key 修复
- 问题：原实现依赖 firstResponder，无输入框时返回失败
- 修复：改用 STHIDEventGenerator 的 HID 事件发送按键
- 新增支持方向键：上(126) 下(125) 左(123) 右(124)
- 新增支持：Home(115) End(119) PageUp(116) PageDown(117)
- 使用 `otherPage:kHIDPage_KeyboardOrKeypad usagePress:` 发送 HID 事件

### /api/reboot 修复
- 原方法在 iOS 上无效
- 修复：使用 notify_post("com.apple.shutdown.reboot") + system("reboot") + kill SpringBoard

### /api/respring 解锁修复
- 原滑动解锁无效
- 修复方式：等待 15 秒 → 按 Home → 等待 1.5s → 再按 Home
- 所有涉及解锁的 API 都已同步更新

### 删除的 API
- `/api/writefile` - Base64 写入文件（已删除）
- `/api/trigger` - 触发懒人精灵（已删除）

### 测试页面和文档同步更新
- test_api.html - 删除懒人精灵 UI 和 trigger API
- API_DOCUMENTATION.md - 删除 writefile base64、trigger 章节
- 添加设备存储信息、按键码详细说明、解锁方式说明

### 新增：文件权限 API
- 解决 Matisu 脚本创建的 700 权限文件夹无法被其他 App 访问的问题
- `GET /api/files/permissions?path=/xxx` - 查看文件/目录权限信息
- `POST /api/files/chmod?path=/xxx&mode=777` - 修改权限
- `POST /api/files/chmod_recursive?path=/xxx&mode=777` - 递归修改权限
- 支持权限颜色标识：🔴700 🟡755 🟢777
- 测试页面添加了文件权限操作面板
