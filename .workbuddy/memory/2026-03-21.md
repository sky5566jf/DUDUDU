### TrollStore 安装功能调试与改进

**问题**: 用户反馈通过 API 安装 IPA 失败，返回 "Installation failed" 错误。

**改进内容**:
1. **改进安装命令执行逻辑**:
   - 优化了 `installAppWithIPAPath` 方法的错误处理
   - 添加了文件可读性检查 (`access(path, R_OK)`)
   - 改进了错误输出，包含更多诊断信息

2. **增强 TrollStore Helper 查找**:
   - 添加了更多可能的 helper 路径（包括越狱环境路径）
   - 改进了动态查找逻辑，支持更多目录结构
   - 添加了日志输出，方便调试

3. **新增诊断 API**:
   - 添加了 `getTrollStoreDiagnostics` 方法
   - 新增 HTTP API `/api/trollstore/diagnostics`
   - 可以检查所有可能的 helper 路径状态
   - 显示找到的 helper 路径和版本信息

4. **更新测试网页**:
   - 添加了 TrollStore 诊断功能
   - 可以一键检查 TrollStore 安装状态

**待测试**: 需要用户在设备上运行诊断 API，查看 TrollStore helper 的实际路径和状态。
