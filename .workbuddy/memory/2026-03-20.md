## 2026-03-20

### 新增 TrollStore 应用管理 API

为 TrollVNC 增加了通过巨魔商店（TrollStore）安装和卸载应用的功能。

**新增 API:**
1. `POST /api/install?path=/xxx/app.ipa` - 通过 TrollStore 安装 IPA 文件
2. `POST /api/uninstall?bundleId=com.xxx.app` - 通过 TrollStore 卸载应用

**修改文件:**
- `src/TVNCApiManager.h` - 添加应用管理 API 声明
- `src/TVNCApiManager.mm` - 实现安装/卸载功能，使用 trollstorehelper 工具
- `src/TVNCHttpServer.mm` - 添加 HTTP 路由和处理函数
- `API_DOCUMENTATION.md` - 更新 API 文档

**实现细节:**
- 自动查找 TrollStore helper 路径（支持多个常见路径）
- 使用 NSTask 调用 trollstorehelper 执行安装/卸载
- 支持错误处理和状态返回
- 需要设备已安装 TrollStore

**使用流程:**
1. 先使用 `/api/upload` 上传 IPA 文件到设备
2. 然后使用 `/api/install?path=...` 安装应用
3. 使用 `/api/uninstall?bundleId=...` 卸载应用
