# 2026-04-03

## TrollVNC 编译修复

### 问题
本地编译报错：`no visible @interface for 'TVNCApiManager' declares the selector 'goToHome'` 和 `openTaskManager`

### 原因
TVNCApiManager.h 头文件中缺少 goToHome 和 openTaskManager 方法声明，但 TVNCApiManager.mm 中有实现

### 修复
在 TVNCApiManager.h 添加 Home 控制 API 方法声明：
- `- (BOOL)goToHome;` - 返回桌面（按一次 Home 键）
- `- (BOOL)openTaskManager;` - 打开任务管理器（双击 Home 键）
